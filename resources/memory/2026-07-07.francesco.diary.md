## 17:27 — Q&A: sovrapposizioni Docker con progetti di lavoro

> Bozza creata da claude. Da rivedere/integrare.

- Chiesto quali punti di ProgettoStudio verranno inevitabilmente in contatto con i Docker di lavoro (UI, IDE, container list).

- Tre buone pratiche fondanti per domani: prefisso `progettostudio-` sui nomi, porte host non-collidenti (MySQL su 3316), reti Docker separate.
